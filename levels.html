🪜 挑戦者レベル設計

挑戦者の「今どこにいるか」を見える化し、最適な支援に繋ぐ仕組み

「何をしたいか分からない（Lv0）」から「他の挑戦者を生み出す側（Lv7）」まで、気持ちではなく観測できる状態で8段階に分類します。
さらに最小7つの質問で自動的にレベルを判定（ルーティング）し、その人に合った支援・成果物・メンターをマッチングします。

📐 8段階のレベルラダー

Lv0
フリーズ
動けない状態。入口の言葉を優しく
※入れない
Lv1
くすぶり
何かしたいがモヤモヤ
3行自己紹介、最初のDMテンプレ
Lv2
一歩目 ★主戦場
最初のアクションへ踏み出す
90日プラン、資金レンジ、巻き込み文面
Lv3
運営見習い
活動を回し始めている
役割分担案、ボランティア募集文
Lv4
実験者
外部との接点を持ち始めている
価値提案、協賛メニュー草案
Lv5
事業化直前
売上/協賛の再現性が見え始め
サービス設計の型、LP/SNS構成
Lv6
自走
自分が抜けても回る
KPI設計、採用要件、外注仕様書
Lv7
拡張
新しい挑戦者を生む仕組みを持つ
展開戦略、投資家向けストーリー

❓ ルーティング判定（最小7問）

#質問選択肢
1直近3ヶ月で活動を実施したか？0回 / 1回 / 複数回
2次回開催日は決まっているか？はい / いいえ
3収入源はあるか？なし / 単発 / 月次
4自分以外の運営担当は？0 / 1 / 2以上
5行政/企業と打合せしているか？0 / 月1未満 / 月1以上
6今の最大の壁は？言語化 / 人手 / 収益化 / マーケ / 制度 / その他
7目指す姿は？継続 / 事業化 / 拡張 / 引き継ぎ
